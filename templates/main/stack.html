    Стек технологий

    Рекомендации по инструментам и окружению для мобильной разработки
    IDE и редакторы
    • Android Studio — полный набор инструментов для Android (Gradle, AVD, Profiler).
    • Xcode (macOS) — разработка под iOS, симуляторы, инструменты профилирования.
    • VS Code — быстрый редактор для Flutter/React Native с богатыми расширениями.
    SDK и фреймворки
    • Kotlin/Jetpack — современная Android‑разработка.
    • SwiftUI — декларативные интерфейсы для iOS.
    • Flutter — кроссплатформенная разработка (Android/iOS).
    • React Native — кроссплатформенная разработка на JS/TS.
    Отладка и тестирование
    • ADB и эмуляторы — лог‑кэт, инсталляция APK/Bundle.
    • JUnit/Mockito (Android), XCTest (iOS) — модульные тесты.
    • Espresso/UIAutomator, XCUITest — UI‑тесты.
    • Charles/Proxyman — перехват и анализ HTTP(S).
    CI/CD и публикация
    • GitHub Actions / GitLab CI — сборки, тесты, линт.
    • Fastlane — автоматизация подписей, скриншотов, публикаций.
    • Play Console / App Store Connect — релизы и тестирование (Internal/TestFlight).
    Настройка окружения
    • JDK 17+ для Android Studio; CocoaPods для iOS.
    • Node.js LTS и nvm для React Native; Flutter SDK с flutter doctor.
    • Recommended linters: ktlint, SwiftLint, ESLint/Prettier.
